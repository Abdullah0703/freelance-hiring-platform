# 🚀 Project Setup Guide  

## 📌 Branching Strategy  
- The **Papa Jani Branch** is named **master**  
- If you need to modify anything, create a new branch using your **own name**  
- Work in your branch and then merge changes when ready  

## 🛠️ How to Run the Project  

### Frontend Setup  
```sh
cd frontend
npm install
npm start
```
### Backend Setup  
```sh
cd backend
npm install
npm start
